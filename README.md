Sales Performance Dashboard (Power BI)

Project Overview

This project presents an interactive Sales Performance Dashboard developed using Microsoft Power BI to analyze business performance across different regions and customer segments. The dashboard helps identify key trends in sales, profit, and quantity over multiple years and supports better data-driven decision making.

Objective

The objective of this project is to explore sales performance patterns, compare segment-wise contributions, and evaluate regional profitability using interactive visualizations.

Dataset Used

The dashboard is created using the Superstore Sales dataset which includes the following key variables:

* Sales
* Profit
* Quantity
* Region
* Segment
* Order Date

Dashboard Features

The dashboard includes multiple interactive visualizations such as:

* Line chart showing yearly trends of Sales, Profit, and Quantity
* Pie chart representing regional distribution of sales performance
* KPI cards displaying Total Sales and Total Profit
* Segment-wise comparison panels
* Region slicer for interactive filtering

Key Insights

* The West region contributed the highest share of overall sales performance.
* Corporate segment showed steady growth across the years.
* Sales and profit trends varied across regions indicating different performance patterns.
* Consumer segment maintained consistent contribution throughout the time period.

Tools and Technologies Used

* Microsoft Power BI
* Data Visualization
* Data Modeling
* Basic DAX Measures

Files Included

Conclusion

This dashboard demonstrates the ability to transform raw sales data into meaningful business insights using interactive visualizations. It highlights key performance indicators and supports effective comparison across regions and customer segments.
